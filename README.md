# Crystals & Dragons

Консольная игра на Swift, выполненная как тестовое задание.
Проект собран без сторонних библиотек: используется только стандартный SDK.

## Что реализовано

### Обязательная часть

- Генерация прямоугольного лабиринта по количеству комнат, которое вводит игрок.
- Гарантированно связная карта (ключ и сундук достижимы из стартовой позиции).
- Описание комнаты в консоли:
  - координаты;
  - количество и направления дверей;
  - список предметов в комнате.
- Команды перемещения: `N`, `S`, `W`, `E`.
- Работа с предметами:
  - `get [item]`;
  - `drop [item]`.
- Открытие сундука командой `open chest` только при наличии ключа в инвентаре.
- Условие поражения: закончился лимит шагов.

### Дополнительная часть

- **Тёмные комнаты**: без `torchlight` игрок видит только сообщение
  `Can’t see anything in this dark place!` и может использовать только движение.
- **Факел**: если бросить `torchlight` в тёмной комнате, она остаётся освещённой.
- **Еда**: команда `eat food` увеличивает доступный лимит шагов.
- **Монстры**:
  - при входе в комнату с монстром выводится предупреждение;
  - у игрока есть 5 секунд на ввод команды;
  - дальше срабатывает вероятностная механика по ТЗ (урон/отброс/успешное действие).
- **Меч**: при наличии меча доступна команда `fight`, успешный исход уничтожает монстра.
- **Золото**:
  - отображается как `gold (N coins)`;
  - собирается через `get gold` с зачислением в счёт монет.
- **Цветной текст** в консоли (ANSI).

## Архитектура проекта

- `siberstest.swift` — точка входа, инициализация игры.
- `GameEngine.swift` — главный игровой цикл и правила.
- `LabyrinthGenerator.swift` — генерация карты, предметов и сущностей.
- `Command.swift` — парсинг текстовых команд.
- `ConsoleIO.swift` — ввод/вывод, цветной текст, ввод с таймаутом.
- `Models.swift` — доменные модели мира.
- `Direction.swift` — направления движения и служебная логика.

Код разделён по ответственности, без «всей логики в одном месте», что соответствует требованию по архитектурному подходу.

## Поддерживаемые команды

- `N`, `S`, `W`, `E`
- `get [item]`
- `drop [item]`
- `eat [item]`
- `open chest`
- `fight`
- `inventory`
- `status`
- `help`
- `quit`

> В командах скобки означают параметр. Например: `get key`, `drop torchlight`, `eat food`.

## Запуск

Из корня проекта:

```bash
swift run
```

Для проверки сборки:

```bash
swift build
```
